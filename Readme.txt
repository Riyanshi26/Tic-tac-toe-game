There are 3 folders for 3 approaches:

1. Q-Learning
    For training the Q-learning agent:
        1. specify the N_episodes number in QPlayer_training.py
        2. run -> python .\QPlayer_training.py
    For Human vs. computer :
        run -> python .\Human_vs_QPlayer.py
    For Human vs. Human :
        run -> python .\Human_vs_Human.py
2. MCTS
    For Human vs. computer :
        run -> python .\run.py

3. SARSA
    For training the agent:
        1. specify the N_episodes number in QPlayer_training.py
        2. run -> python .\QPlayer_training.py
    For Human vs. computer :
        run -> python .\Human_vs_QPlayer.py
